# playbooks/

Operative Standards. Wie Mala welche Sache standardmässig macht. Jeder Playbook ist eine Datei mit klarem Schritt-für-Schritt, der von jedem Team-Member umgesetzt werden kann.

## Konvention

Jeder Playbook hat:

1. **Wann nutzen:** Trigger-Bedingung
2. **Owner:** wer ist verantwortlich
3. **Schritte:** nummeriert, mit klaren Outputs pro Schritt
4. **Done-Kriterium:** wann ist der Playbook durch
5. **Eskalation:** wann was an wen weiterleiten
