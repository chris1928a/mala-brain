# 05 Meetings

Alle Meeting-Notes. Strukturierte Versionen plus Raw-Transkripte.

## Konvention

- Strukturierte Meeting-Notes: `YYYY-MM-DD-thema.md` direkt in diesem Folder
- Raw-Transkripte (Gemini, Otter, manuell): `raw/YYYY-MM-DD-source.md`
- Owner: wer im Meeting war, einer übernimmt das Strukturieren

## Format strukturierter Notes

```markdown
# Thema, Datum

Teilnehmer:
Anlass:

## Zusammenfassung
## Decisions
## Action Items pro Owner
## Kontext-Updates
## Nächster Termin
```
